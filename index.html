Hand Gesture Detector
Tools/Languages: Python, Tensorflow, MediaPipe, OpenCV
Repository: Link
By capturing and training on landmark data from my hand, I was able to make a ML model that could predict a hand gesture captured on a webcam. The model can also be extended to recognize custom hand gestures. Curently there are no further projects planned to incorporate this, but it certainly could be implemented somewhere in the future. More details on the specific implementation and training can be found in the Github link above.
Pixelator
Tools/Languages: Java, Swing
Repository: Link
Relatively simple program made as the summative project for Gr.12 Computer Science. As input it takes a regular photo/image then attempts to convert it into pixel art through an algorithmic pipeline. The user also gets to choose between 3 processing algorithms and pixel dimensions. While the end result is not necessarily complicated, the program successfully implements some key concepts such as OOP, threading, data structures (maps, arrays, lists, etc), and custom algorithms.

Tools/Languages: JavaScript, Node.js, MongoDB, Heroku, Various APIs and libraries
Repository: Link
The codebase for this project makes me want to go blind as it was one of the first "large (perhaps largest?)" personal projects I ever got into.
